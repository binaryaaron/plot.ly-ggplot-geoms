Plotly

Plotly is a cool, open-source project ( github ) to help people get interactive plots online easily through a number of various programming languages. It's essentially brand new and I'm going to document how well it works with the popular ggplot2 package in R. This document will be updated as support for more geoms is added.

Plotly and geoms

I'm going to test out different ggplot geoms to check for compatibility. Most examples will be taken straight from ggplot's docs website. The master list of geoms is at the botttom of this post. Some of the plots have minor errors.
